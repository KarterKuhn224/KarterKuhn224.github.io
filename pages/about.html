About This Project

Problem → Promise: Scoreboards can be heavy or fixed in place, which makes setup slow and limits where you can play. Our concept is a simple, movable option that keeps games running smoothly.

The website is intentionally minimal to match a level-1 rubric: one CSS file, clean navigation, mobile-friendly meta tag, and a small media query.
